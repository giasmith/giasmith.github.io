User Stories

User roles

  • Renters
    • Users who need to be able to search for a desired item, post a request for an item, communicate with potential lenders, and bookmark items so they can return to it later.
  • Lenders
    • Users who need to be able to post items they are interested in lending out, search for requested items, communicate with renters, and set a desired area to lend in.
  • Administrators
    • Users who can monitor user posts and conversations and can enforce website policy.

As a Renter, I want to...

  • Search for items I wish to rent, so I can more easily find what I want.
  • Pay for rented items online or in person, so that I can choose what is more convenient to me.
  • Search for lender in my area, so it is more convenient to obtain a desired item.
  • Bookmark items, so that I can easily view them later.
  • Be able to communicate with lenders.
  • Be shown the condition of the item I am renting, so I can determine if it is worth the price.
  • Be able to rate Lender accounts.

As a Lender, I want to...

  • Post items I wish to lend, so that they may be viewed by potential renters.
  • Be able to post how long I am willing to rent an item and the price I will rent it for.
  • Select the area I intent to lend in, so that I can find potential renters near me.
  • Receive payment either online or in person, so I can choose what is more convenient to me.
  • Be able to communicate with renters.
  • Be compensated for any damage that an item obtains while being rented.
  • Be able to rate Renter accounts.

As an Administrator, I want to...

  • Enforce website policy, so that I can remove posts or users that violate it.
  • View reported posts, so that I can enforce website policy.
  • Push security and version updates to the live site.
